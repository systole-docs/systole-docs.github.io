    systole.plot_raw¶

    systole.plot_raw(signal, sfreq=75)[source]¶

    Interactive visualization of PPG signal and beats detection.

    Parameters
    signalpd.DataFrame instance or 1d array-like

    Dataframe of signal recording in the long format. Should contain at least the two following columns: [‘time’, ‘signal’]. If an array is provided, will automatically create the DataFrame using th array as signal and sfreq as sampling frequency.

    sfreqint

    Signal sampling frequency. Default is 75 Hz.
